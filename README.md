# ChatBot Acessível em Python

Este projeto é um **piloto de estudo em Python** com foco em **aprendizado prático da linguagem** e na criação de um **chat interativo acessível**.  
A ideia é explorar bibliotecas gratuitas e poderosas, integrar **IA** e **reconhecimento de voz**, e construir uma base sólida para projetos maiores no futuro.

## Objetivos
- Fixar conceitos fundamentais de **Python**.
- Explorar integração com **IA** (NLP e modelos de linguagem).
- Implementar **entrada por voz** e **saída em áudio** para acessibilidade.
- Criar uma aplicação **web interativa** com Flask.
- Evoluir para um projeto escalável e útil para a comunidade.

## Tecnologias utilizadas
- **Flask** → back-end web simples e rápido.
- **HTML, CSS, JavaScript** → interface do chat.
- **SpeechRecognition + PyAudio** → reconhecimento de voz.
- **gTTS (Google Text-to-Speech)** → respostas em áudio.
- **Transformers (Hugging Face)** → integração com modelos de IA gratuitos.
- **SQLite / SQLAlchemy** → persistência de dados e histórico de conversas.


## Features Futuras

Este projeto é um piloto de estudo, mas já tem planos de expansão para se tornar uma aplicação acessível e robusta. Algumas ideias de evolução:

- **Integração com reconhecimento de voz (SpeechRecognition + Whisper)**  
  Permitir que usuários interajam por voz e recebam respostas faladas.

- **Síntese de voz (gTTS ou TTS local)**  
  Tornar o chatbot inclusivo para pessoas com deficiência visual.

- **Autenticação de usuários**  
  Integração com [py_system_login](https://github.com/DevNinee/py_system_login) para login seguro.

- **Persistência de conversas**  
  Armazenar histórico em banco de dados (SQLite/PostgreSQL) para análise posterior.

- **Dashboard de dados**  
  Criar visualizações com **Streamlit** ou **Dash** para estudar estatísticas das conversas.

- **IA avançada com Hugging Face Transformers**  
  Substituir ou complementar respostas com modelos de linguagem open-source.

- **Modo acessibilidade**  
  Ajustes de contraste, tamanho de fonte e navegação por teclado para inclusão digital.

- **Hospedagem em nuvem**  
  Disponibilizar o chatbot online (Heroku, Render, AWS) para acesso público.

- **Expansão para Rasa ou FastAPI**  
  Evoluir para frameworks mais robustos e escaláveis.
